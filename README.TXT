Advent of Code Solutions 2021
=============================

This year I started out being nostalgic, so I started out in QBasic, the first language I learned in anger.

This quickly got infuriating as I started to hit the limits of the language and was spending all of the time just implementing basic functions that I would nowadays expect to be standard library type functions.

So I gave up and started over in Ruby, because I was missing out on the actual fun of the puzzles.
